# Economy Market Simulator

An agent-based simulation of a simple economy with Households and Firms.

## How to Run

```bash
python EmulationModeling/41_economy_market_simulator/main.py
```

## Logic

1.  **Households**: Work for firms, earn wages, and buy goods.
2.  **Firms**: Hire households, produce goods based on labor, sell goods, and adjust prices/wages based on supply/demand.
3.  **Market**: Prices fluctuate based on scarcity.
4.  **Visualization**: Plots trends of average price/wage and wealth distribution histogram.
