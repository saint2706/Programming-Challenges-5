# Top-Down Shooter

A top-down shooter game implemented in Python using Pygame.

## Features
- **Controls:** WASD to move, Mouse to aim and shoot (Left Click).
- **Enemies:** Red squares that chase the player.
- **Combat:** Shoot enemies to destroy them.
- **Stats:** Health system and Score tracking.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **W, A, S, D:** Move Player
- **Mouse:** Aim
- **Left Click:** Shoot
- **Esc:** Quit
