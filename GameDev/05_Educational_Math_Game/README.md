# Educational Math Game

A simple math quiz game implemented in Python using Pygame.

## Features
- **Gameplay:** Solves random math problems (Addition, Subtraction, Multiplication).
- **Difficulty:** Numbers get larger as your score increases.
- **Input:** Type the answer using the keyboard.
- **Feedback:** Visual feedback for correct/incorrect answers.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **Number Keys:** Type answer
- **Backspace:** Delete last digit
- **Enter:** Submit answer
- **Esc:** Quit
