# Breakout / Arkanoid Clone

A classic brick-breaking game implementation in Python using Pygame.

## Features
- **Physics:** Ball bouncing off walls, paddle, and bricks.
- **Controls:** Move paddle with Mouse or Left/Right Arrow keys.
- **Game States:** Start Screen, Playing, Game Over (Win/Lose).
- **Lives System:** Player starts with 3 lives.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **Arrow Keys / Mouse:** Move Paddle
- **Space:** Launch Ball (if stuck to paddle)
- **Enter:** Start Game / Restart
- **Esc:** Quit
