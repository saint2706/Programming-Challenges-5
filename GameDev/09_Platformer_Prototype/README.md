# Platformer Prototype

A basic platformer game engine implemented in Python using Pygame.

## Features
- **Physics:** Gravity, jumping, collision detection (AABB).
- **Level:** Platforms, hazards (spikes), and a goal.
- **Goal:** Reach the green portal to win.
- **Fail:** Hitting a red spike resets the level.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **Left/Right Arrow:** Move
- **Space:** Jump
- **R:** Restart
- **Esc:** Quit
