# Typing Game

A typing practice game implemented in Python using Pygame.

## Features
- **Gameplay:** Words fall from the top of the screen. Type them before they hit the bottom!
- **Feedback:** Highlights the part of the word currently matched.
- **Difficulty:** Words fall faster as you progress.
- **Lives:** Miss 3 words and it's Game Over.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **Keyboard:** Type the active word.
- **Enter:** Restart (on Game Over)
- **Esc:** Quit
