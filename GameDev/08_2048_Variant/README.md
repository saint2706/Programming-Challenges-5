# 2048 Variant

A 2048-style puzzle game implemented in Python using Pygame.

## Features
- **Gameplay:** Slide tiles on a 4x4 grid. Same numbers merge into larger ones.
- **Goal:** Reach the 2048 tile!
- **Score:** Points awarded for every merge.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **Arrow Keys:** Slide tiles (Up, Down, Left, Right).
- **R:** Restart
- **Esc:** Quit
