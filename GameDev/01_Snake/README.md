# Snake with Polished UX

A classic Snake game implementation in Python using Pygame.

## Features
- **Smooth Controls:** Responsive arrow key movement.
- **Score System:** Track your current score and high score.
- **Persistence:** High scores are saved locally to `highscore.txt`.
- **Game States:** Start Screen, Playing, Paused, Game Over.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **Arrow Keys:** Move Snake
- **P:** Pause / Resume
- **Enter:** Start Game / Restart (on Game Over)
- **Esc:** Quit
