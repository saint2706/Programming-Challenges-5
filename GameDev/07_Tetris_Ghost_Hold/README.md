# Tetris with Ghost Piece & Hold

A Tetris-style block stacking game in Python using Pygame.

## Features
- **Gameplay:** Standard tetrominoes, line clearing, gravity.
- **Ghost Piece:** Shows where the current piece will land.
- **Hold Function:** Swap the current piece with a held piece (press C).
- **Controls:** Arrow keys for movement/rotation, Space for hard drop.

## How to Run

1.  Ensure you have Python installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
- **Arrow Keys:** Move Left/Right/Down
- **Up Arrow:** Rotate
- **Space:** Hard Drop (Instant)
- **C:** Hold Piece
- **Esc:** Quit
