## Setup

1. Get your free API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Replace the API_KEY in `src/main.js` with your key
3. Install and run:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
