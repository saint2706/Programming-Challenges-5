# System Health Dashboard

A dashboard to monitor system health metrics like CPU usage, memory, and disk space.

## Requirements

- Python 3.8+
- `psutil`
- `rich`

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python "Practical/System Health Dashboard/dashboard.py"
```

## Features

- Real-time CPU usage visualization.
- Memory usage stats (RAM + Swap).
- Disk partition usage.
- Network I/O stats.
- Updates every second.
